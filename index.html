Projects

A curated selection of my technical work, demonstrating applied expertise in data science, machine learning, and software development. Each project reflects problem-solving rigor, iterative refinement, and cross-functional collaboration—delivering measurable impact through code, algorithms, and analytical insights.

Mar 2025

Bachelor Thesis: Multimodal RAG Evaluation for Enterprise Knowledge Retrieval

As part of a collaboration with Primeo Energie and my University, I developed and evaluated a multimodal Retrieval-Augmented Generation (RAG) system. The goal was to enable accurate retrieval and summarization of complex enterprise documents—including text and tables—to support internal knowledge sharing. I implemented reranking strategies, context-aware evaluation metrics, and conducted performance analysis using tools like RAGAS and custom benchmarks.

Jan 2025

Explainable AI Project: Understanding CNN Decisions on MNIST Variant

I explored the decision-making process of a convolutional neural network trained on a variant of the MNIST dataset. Using explainable AI techniques such as Vanilla Gradient Saliency, SmoothGrad, Integrated Gradients, Grad-CAM, and Occlusion, I visualized which pixels influenced the model’s predictions. The project focused on interpreting CNN behavior, identifying data issues, and discussing the limitations of XAI methods, all documented in a clean, interactive Jupyter Notebook.

Dec 2024

NLP Project: Sentiment Analysis

In this NLP project, I explored how varying levels of labeled data affect sentiment classification performance. I trained a baseline transformer model and enhanced it using weak labeling techniques based on semantic text embeddings. The project included data analysis, hierarchical splitting, model comparison, and embedding visualization. Final evaluation was conducted using accuracy, precision, recall, and F1-score to assess improvements from data augmentation.

Jun 2024

Deep Learning Project: Image Classification

In this project, I trained and evaluated several neural network models for image classification using the CIFAR-10 dataset, which contains 60,000 color images across 10 object classes (e.g., cars, birds, ships). The focus was on comparing model performance through systematic experimentation. I used the MLOps platform Weights & Biases to monitor and analyze training behavior in real-time.

Jun 2024

Machine Learning Project: Real Estate Predictor

In this challenge, I worked with a dataset of Swiss real estate properties to develop machine learning models for predicting property prices and classifying object types. The project included exploratory data analysis, linear regression with residual analysis, and advanced regression modeling submitted as part of a Kaggle-style competition. Additionally, I built a simple web service to serve predictions and compared classification performance for property types.

Jan 2023

NLP Project: Tinder 4 Movies

The goal for this project was to create a movie recommendation system based on similar movies and user's preferences (contect based and collaborative systems). The system was built using Python and the data was scraped from the TMDB API and MovieLens database.

Jun 2022

Machine Learning & Data Analysis Project: Influence of water temperature on coffee taste

My team and I received data from a professional coffee barista and analyzed the data to find out if there is a correlation between water temperatures and coffee tastes for specific machines and roasting techniques.

Jun 2023

Data Analysis Project: Policy Advice on Energy Production without Greenhouse Gas Emissions

My team and I used Data Science to analyze the energy production in Switzerland and other surrounding countries and give policy advice on how to produce energy without greenhouse gas emissions.

Apr 2022
